Welcome to Rest API Examples:-

Following are the operation you can perform on the API

  • Create

    • To add an customer please provide name and email of the customer in the Html body and sent the post request e.g.

    URI : https://restcrud.abhishekkumarsingh.now.sh/customer

    { "name":"John Doe", "email":"johndoe@xyz.com" }

  • Retrive

    • To retrive the data from the database use the below URI and sent the post request

    URI : https://restcrud.abhishekkumarsingh.now.sh/customer?email="emailId"

  • Update

    • To update the Customer Data you can use PUT request with the name and email attributes in the body of the request.

    URI : https://restcrud.abhishekkumarsingh.now.sh/customer

    Body should be like:

    { "name":"John Doe", "email":"johndoe@xyz.com" }

  • Delete

    • To delete the perticular user pass the user email in the query parameter with delete request

    URI : https://restcrud.abhishekkumarsingh.now.sh/customer?email="emailId"
